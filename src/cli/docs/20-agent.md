# Agent Commands

Agents are discoverable identities exposed by Spiderweb over the unified-v2 control plane.

## agent list

List available agents.

**Examples:**
```bash
spider agent list
```

## agent info <agent_id>

Show details for one agent.

**Arguments:**
- `agent_id` - Agent identifier (for example `spiderweb`)

**Examples:**
```bash
spider agent info spiderweb
spider agent get spiderweb
```
